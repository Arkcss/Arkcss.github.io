Features

Ark is a framework that fixes many common issues
from other options. It contains a smart flex grid,
that will help you achieve your objectives really fast.

Flex Grid

That works perfectly.

This grid doesn't pop holes. A row is a row, and a column is a column, there are no weird nesting issues.

Components

Easy to implement.

Write the components options as html tag attributes. Only include the framework JS.

Style

Easy to customize.

Style adjustable in SASS files. Source files properly organized.
